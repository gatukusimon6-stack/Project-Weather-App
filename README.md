# Weather Dashboard App

This is a simple Weather Dashboard built using HTML, CSS, and JavaScript.  
It fetches real-time weather data from the OpenWeatherMap API using asynchronous JavaScript.

---

##  Description

This project allows users to search for any city and view current weather information including:

- Temperature 
- Weather conditions  
- Humidity   
- Wind speed   

It demonstrates key JavaScript concepts like:
- Async/Await  
- Fetch API  
- Error Handling  
- DOM Manipulation  
- Loading States  

---

## Features

- Search weather by city name
- Real-time API data fetching
- Loading indicator while fetching data
- Error handling for invalid input or cities
- Clean and simple UI updates

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- OpenWeatherMap API
