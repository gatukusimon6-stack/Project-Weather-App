## Project Description
This project allows users to search for any city and view current weather information including:

- Temperature 
- Weather conditions  
- Humidity   
- Wind speed   

It demonstrates key JavaScript concepts like:
- Async/Await  
- Fetch API  
- Error Handling  
- DOM Manipulation  
- Loading States
